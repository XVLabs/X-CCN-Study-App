# X-CCN-Study-App

The X-Computer Communications and Network Study or X-ITC Study for short!

This mobile app is a quiz app designed to aid students with the Computer Communications and Networking module, teaching them the fundamentals of networks.

It covers eight chapters

Data Communications Fundamentals
Networking Models & Architecture
Physical Layer & Transmission Media
Data Link Layer
Network Layer & IP Addressing
Transport Layer
Application Layer & Protocols
Networking Security & Management

You are also able to review and recheck your scores after each attempt, BUT you don't get to see the right answers. This is an attempt to make students continously try to understand the concepts and demonstrate them afterwards in exams or other situations that may need this knowledge.
